# shape-missing-tradeoffs

Options A and B each list only upsides — "best option", "easiest path"
— and cite zero costs, risks, or trade-offs. The Shape Coherence
rubric (criterion 4) requires every substantive option to name at
least one concrete downside so the picker can make an informed
selection. Defer (Option C) is fine, but A and B must RETRY with
`missing-tradeoffs`.

Keyword: missing-tradeoffs
