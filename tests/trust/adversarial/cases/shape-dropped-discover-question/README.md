# shape-dropped-discover-question

The Discover brief raises three open questions: per-org vs per-user
aggregation, Stripe Connect vs our own ledger, and whether to
grandfather existing Pro subscribers. Options A and B pick positions
on the first two, but no option addresses the grandfathering
question — not picked, not punted. The Shape Coherence rubric
(criterion 5) requires the proposal to reconcile every open question
the Discover brief named.
