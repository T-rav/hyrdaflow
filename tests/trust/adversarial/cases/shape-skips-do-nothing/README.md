# shape-skips-do-nothing

The proposal lists three substantive options (structlog, home-grown
formatter, partial adoption) and each names a trade-off — but there
is no "Defer" / "No-op" / "accept status quo" option anywhere. The
Shape Coherence rubric (criterion 2) requires an explicit do-nothing
option naming the cost of inaction, so the picker can consciously
choose to not spend the engineering time.

Keyword: missing-defer
