# discover-missing-acceptance-criteria

The Discover Completeness rubric (criterion 1 — Structure) requires all
five named sections: *Intent*, *Affected area*, *Acceptance criteria*,
*Open questions*, *Known unknowns*. This brief omits the *Acceptance
criteria* section entirely, so the evaluator must RETRY with the
`missing-section:acceptance-criteria` keyword before downstream phases
are allowed to run.

Keyword: missing-section:acceptance-criteria
