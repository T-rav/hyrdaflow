# shape-options-overlap

Option A and Option B both touch `src/search/query.py` and
`src/search/index.py` with trivially different wording — the only
visible difference is which SaaS vendor gets namedropped. That is 100%
scope overlap, well past the rubric's 50% threshold. The Shape
Coherence rubric (criterion 3) requires mutually exclusive option
scope so the human picker is choosing between real alternatives, not
two coats of paint on the same plan.
