# discover-vague-criterion

All five rubric sections are present with substantive content, but the
*Acceptance criteria* bullets are aspirations, not observables — no
metric ("faster" but not "p95 under 200ms"), no signal ("happier" but
not "NPS ≥ 40" or "weekly active ≥ X"), no measurable surface
("love it" but nothing to inspect). The Discover Completeness rubric
(criterion 4) requires every acceptance bullet to name an observable
outcome.
